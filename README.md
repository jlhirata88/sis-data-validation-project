# Student Information System Data Validation & Reporting Project

## Overview
This project simulates a Student Information System (SIS) environment using SQLite and Python. It was built to demonstrate systems analysis, data validation, SQL reporting, and problem-solving skills relevant to Systems Analyst roles.

## Tools Used
- Python
- SQLite
- PyCharm
- GitHub

## Features
- Creates a relational database with students, courses, and enrollments
- Identifies missing student emails
- Detects duplicate student names
- Finds missing grade records
- Generates student performance reports
- Calculates average grades per student
- Exports report data to CSV

## Files
- `create_database.py` - Builds and populates the database
- `run_reports.py` - Runs validation and reporting queries
- `export_report.py` - Exports student report data to CSV
- `school.db` - SQLite database file

## Sample SQL/Data Validation Tasks
- Find null values in key fields
- Identify duplicate records
- Join multiple tables for reporting
- Validate reporting outputs for data accuracy
